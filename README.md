# Indoor-Map



使用[Indoor-Map-Draw](https://github.com/WoShiSunWuKong/Indoor-Map-Draw) 这个工具来绘制楼宇房屋的坐标，坐标文件用geojson格式来保存

### 用法
把代码拉下来
```
执行npm install  安装依赖文件
```
 
```
安装好之后，运行 npm start
```

```
编译好后，浏览器会自动打开demo
```
